# Minivibes — Enclose clone (prototype)

Prototype pour recréer le jeu Enclose en utilisant Flask.

Exécution en local :

```powershell
python -m venv env
env\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Où :
- `app.py` lance l'application Flask
- Les templates sont dans `templates/` et les assets dans `static/`

Prochaine étape : initialiser le dépôt Git et valider ces fichiers.
